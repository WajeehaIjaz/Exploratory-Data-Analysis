# Student EDA Project 📊

This project performs Exploratory Data Analysis (EDA) on a student dataset to understand relationships between CGPA, attendance, grades, and other academic features.

## 📁 Dataset Features
- name  
- course  
- cgpa  
- attendance  
- grade  
- results  
- scaled_cgpa  

## ⚙️ Tools Used
- Python  
- Pandas  
- NumPy  
- Seaborn  

## 📊 Analysis Performed
- Loaded and explored dataset  
- Checked summary statistics and missing values  
- Removed unnecessary column (`scaled_cgpa`)  
- Performed correlation analysis  
- Visualized data using:
  - Heatmap (correlation)
  - Pairplot (relationships)
  - Relplot (Attendance vs CGPA by Grade)
  - Distplot (CGPA distribution)
  - Boxplot (outlier detection)

## 📌 Key Insight
Visual analysis helps understand student performance patterns based on attendance and CGPA.
